[size=4][color=#bf307a]Introduction[/color][/size]
[size=1em]Alternate User Posting mod will allow you to post as a different user or change the user linked to a post, providing you have the right permissions to do so. Each user with the correct permissions will see an additional field to place a the name of another member to be the poster of the message.[/size]

[size=3][color=#1f4882]Features[/color][/size]
[list]
[li][size=0.9em]Set permissions for who can post as an alternate user/change the author of a post.[/size][/li]
[/list]
